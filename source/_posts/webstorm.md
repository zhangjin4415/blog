第一章 常见问题
=====================
1.打开网页中文乱码
-------------------
方法1：
你页面里面
```
<meta charset="utf-8">
```
方法2：
file ----->  settings 搜索encoding
在fileencoding下，将IDE Encoding，Project Encoding和下面的Default Encoding for properties file都更改为utf-8。
